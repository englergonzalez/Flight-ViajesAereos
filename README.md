## Instalación

    1. git clone https://github.com/englergonzalez/Flight-ViajesAereos.git
    2. composer install
    3. copy .env.example .env
    4. php artisan key:generate
    5. Configurar el archivo .env
        1. DB_PORT=
        2. DB_DATABASE=
        3. DB_USERNAME=
        4. DB_PASSWORD=
    6. php artisan migrate:fresh --seed
    7. npm install
    8. npm run dev
    9. php artisan serve
